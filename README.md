# freshcartreactjs
# Ecommerce React JS Project

## What is the use of this Repo

This Project is a Ecommerce ReactJS Project which demonstrates the following
1. Creating a Component in React
2. Making HTTP calls 
3. Communicating between parent and child component
4. Using Bootstrap along with React and use react-bootstrap
5. Using Basic Routing in React
6. Authentication and Authorization
7. online Payment
8. API

## Used Technologies
1. React js
2. javaScript
3. Context Api
4. React-router-dom
5. react-Query
6. react hooks( useState-useContext-useparam-useEffect)
7. useformik

## Prerequisites

### Install Node JS
Refer to https://nodejs.org/en/ to install nodejs

### Install create-react-app
Install create-react-app npm package globally. This will help to easily run the project and also build the source files easily. Use the following command to install create-react-app

```bash
npm install -g create-react-app
```

## Cloning and Running the Application in local

Clone the project into local

Install all the npm packages. Go into the project folder and type the following command to install all npm packages

```bash
npm install
```

In order to run the application Type the following command

```bash
npm start
```



## Application design

#### Components

1. **Products** Component : This Component displays a list of Products.

2. **Category** Component : This Component Displays the Category Of products. 

3. **Brands** Component : This Component Displays the Brands of Products. 

4. **Cart** Component : This Component Displays last added products to the cart to purchase it.
  
5. **AllOrders** Component : This Component Displays orders that you purchase.
   
6. **Payment** Component : This Component make offline payemt or online payment.

7. **Profile** Component : This Component show you Welcome message.
  
 




#### HTTP client

**axios** library is used to make HTTP Calls

#### URL

The application has just one url /customerlist which ties to *Customers* Component
